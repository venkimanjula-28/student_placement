# student prediction system
This Project Predicts Whether a student is placed based on Academic Performance And skills

## features
~Data Preprocessing
~Machine Learning model training
~StreamLit Prediction
~Real-Time Prediction

## Project Structure
Student-Placement/
                data/
                    Sample.csv
                notebooks/
                    eda.ipynb
                src/
                    train_model.py
                    predict.py
                app.py/
                requirements.txt
                README.md

## how to run
1. Install All Dependencies
    pip install -r requirements.txt
2. Train Model
   cd src/Train_model.py
3.Run the Streamlit app
   streamlit run app.py
